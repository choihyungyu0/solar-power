# 01. 서비스 개요 — 우리아파트 태양광 설치하기

## 한 줄 정의

경기도 공동주택·건물주가 태양광 설치 가능성, 예상 절감액, 정책자금 후보, 신청 알림을 한 번에 확인하는 도심형 태양광 도입 지원 서비스.

## 문제 정의

1. 전기요금 상승으로 공동주택 공용부·건물 운영비 부담이 커진다.
2. 태양광은 자가발전 이익이 있지만, 설치 가능성·수익성·규제·보조금 정보를 주민/건물주가 직접 판단하기 어렵다.
3. 정책자금은 공고 기간과 조건이 자주 바뀌어 놓치기 쉽다.
4. 지자체는 태양광 예산을 효과적으로 소진하고 실제 설치까지 이어지는 대상지를 찾기 어렵다.

## 핵심 사용자

- 아파트/공공주택 입주자대표회의
- 관리사무소/관리주체
- 건물주/토지주
- 경기도 및 시군 에너지 정책 담당자
- 태양광 시공/금융 연계 사업자

## 핵심 기능

### 1. 우리 아파트 태양광 설치하기

입력값:

- 주소/단지명
- 건물 유형
- 세대수
- 옥상 면적
- 월 전기요금
- 음영 점수 또는 일조 조건

결과값:

- 설치 적합도
- 추천 설치용량
- 예상 발전량
- 예상 전기요금 절감액
- 정책지원 후보
- 회수기간
- 다음 액션 체크리스트

### 2. 태양광 가상설치

모햇 사이트의 서비스 소개처럼 사용자가 “내 건물에 설치하면 어떤 그림이 나오는지”를 빠르게 이해하도록 구성한다.

MVP에서는 3D 정밀 구현 대신 아래 방식으로 시연한다.

- 도심형 카드/옥상 패널 시각화
- 입력값에 따른 적합도 게이지
- 예상 설치용량/발전량 카드
- 정책자금 후보 카드

고도화 시:

- 지도 기반 옥상 선택
- 항공사진/건축물대장/DSM 기반 옥상 면적 추정
- 음영/일사량 기반 패널 배치
- 규제지역 레이어 표시

### 3. 정책자금 안내

정책자금 금액은 연도·시군·공고별로 달라지므로 MVP에서는 “후보 정책 + 공고 확인 필요 + 알림 신청” 구조로 설계한다.

정책자금 카드 예시:

- 경기도 주택태양광 지원
- 한국에너지공단 주택지원
- 베란다형 미니태양광
- 금융지원/담보대출 연계 가능성

### 4. 알림

실시간 API가 없어도 서비스 가치는 유지된다.

- 카카오톡 알림톡
- SMS
- 이메일
- 웹 알림

알림 주제:

- 정책 공고 시작
- 예산 소진 임박
- 서류 누락
- 리포트 생성 완료
- 설치 가능성 재검토 필요

### 5. 회원가입/로그인

회원 기능은 다음을 위해 필요하다.

- 리포트 저장
- 정책자금 알림 신청
- 신청서류 진행상태 관리
- 여러 건물 비교
- 입주자대표회의 공유 링크 발급

## 서비스 메시지

1. 전기요금은 계속 부담이 커진다.
2. 태양광 자가발전은 공용 전기요금 절감 수단이 될 수 있다.
3. 보조금과 정책자금은 우리가 찾아주고, 신청 타이밍을 알려준다.
4. 사용자는 우리 사이트에서 예상 혜택과 비용을 먼저 확인한다.

## MVP 화면 구성

1. 랜딩/서비스소개
2. 우리 아파트 태양광 설치하기
3. 시뮬레이션 결과
4. 정책자금 후보
5. 알림 신청
6. 가입후기
7. 회원가입/로그인

## 심사/공모전 관점 어필

- 공공성: 경기도 태양광 예산 소진 지원, 에너지 전환 참여 확대
- 시의성: 전기요금 부담, 탄소중립, 도심 전력수요 이슈
- 실현가능성: 기존 공공데이터와 정책 사이트를 활용한 MVP 구현 가능
- 사업성: B2G 예산소진 솔루션 + B2C/B2B 설치 리포트 + 금융/시공 연계
